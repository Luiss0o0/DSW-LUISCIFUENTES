Contenido en HTML

serie de elementos que usarás para encerrar diferentes partes del contenido para que se vean o comporten de una determinada manera. Las etiquetas de encierre pueden hacer de una palabra o una imagen un hipervinculo a otro sitio, se pueden cambiar palabras a cursiva, agrandar o achicar la letra, etc. A continuacion se presentan las etiquetas mas usadas en HTML:

Etiquetas principales y su funcion

  • < !DOCTYPE>: Define el tipo de docuemento

  • < article>: Define un artículo

  • < body>: Define el cuerpo del documento

  • < canvas>: Se usa para dibujar gráficos en pantalla

  • < div>: Define una sección en un documento

  • < footer>: Define el pie de página de un documento

  • < form>: Define un formulario html

  • < h1> a < h6>: Define encabezados o títulos

  • < head>: Define información hacerca del documento

  • < header>: Define la sección de encabezado del documento

  • < html>: Define la raíz del documento

  • < img>: Define una imagen
