      Contenido en HTML

      Es una serie de elementos que usarás para encerrar diferentes partes del contenido para que se vean o comporten de una determinada manera. Las etiquetas de encierre pueden hacer de una palabra o una imagen un hipervinculo a otro sitio, se pueden cambiar palabras a cursiva, agrandar o achicar la letra, etc.
